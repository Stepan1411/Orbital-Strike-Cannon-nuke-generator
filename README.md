# Orbital-Strike-Cannon-nuke-generator
This script generates a nuke like Wemmbu

How to use

=== TNT Sphere Generator ===

Number of TNT circles: (number of rings in the charge)

Customize TNT count in circles? (y/n, default=n): (you can change the number of TNT in a circle)

Spawn height offset (e.g., 0, 1.5, -2, default=0): (height from the player where to spawn the charge)

Outward speed (inner circle): (speed of the first circle)

Outward speed (outer circle): (speed of the last circle)

Vertical boost: (the speed at the spawn point, can be negative)

Fuse time in ticks (default=80): (setting the TNT explosion time in ticks)

=== Generated ??? commands ===

Save to file? (y/n): (save the file in the settings with the script)

Filename (default=tnt_commands.txt): (file name)
